JavaScript Loops and Functions

Using functions in JavaScript, do the following:

  • Part 1:
    1. In your JavaScript file, write the following: greeting("ThisName"). Replace ThisName with your actual name
    2. Make a function called "greeting". This function will take one parameter called "yourName"
    3. In your function, make a for-loop that starts at 0, ends at 10 (less than 10), and increases by 1 every time the loop runs
    4. In your loop, print to the console "Hello, my name is " + yourName
    5. When you're done, you should see it say "Hello, my name is [yourName]" 10 times
  • Part 2:
    1. In your JavaScript file, write the following: console.log(pythagorean(3,4,5))
    2. Make a function called "pythagorean" with 3 parameters called x, y, and z
    3. To calculate the sides of a right triangle, we use the Pythagorean Theorem which states x² + y² = z²
    4. In our function, return true if x² + y² = z² and return false if x² + y² ≠ z². To get something like x², you can do x * x
    5. When you're done, you should see true in your console with the code I've provided
