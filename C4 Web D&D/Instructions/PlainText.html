Making HTML, CSS and JS Files on Our Computers

Many programmers code on their local computer without needing something like Repl.it. Here, I'll help you transfer your Repl.it website to your PC:

Apple

  1. Make a folder to save all of your stuff
  2. Open TextEdit and change your empty file into a plain text file (.txt) through the format tab
  3. Copy and paste your HTML, CSS, and JS code into 3 separate files and save those files into your folder with the same name as your files on Repl.it.

Windows

  1. Make a folder to save all of your stuff
  2. Open Notepad and copy and paste your HTML, CSS, and JS code into 3 separate files.
  3. Save those files into your folder with the same name as your files on Repl.it
